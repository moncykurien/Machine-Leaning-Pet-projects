General Properties¶

Observations from the Dataset:


  1. The fields 'budget','revenue','budget_adj' and 'revenue_adj' are currencies. We can assume that they are in dollars since thats not specified.
  2. Number of votes per movie is different for each movie.i.e., the vote-count has different values. So we cannot use vote-count or vote_average to gauge the success/likeness of a movie.
  3. The release year ranges from 1960 to 2015.
  4. The dataset contains rows that have budgets and/or revenue as 0. These are missing data. Needs cleaning.
  5. Any duplicate rows must also be cleaned(rows 2089 and 2090 are duplicates.
  6. The unnecessory columns such as ` 'id', 'imdb_id', 'popularity', 'budget_adj', 'revenue_adj', 'homepage', 'keywords', 'overview', 'production_companies', 'vote_count' and 'vote_average' can be removed.
  7. And date columns must be converted into date format.

def get_count(dataset,col,symbol,threshold):
    '''
    This function gets the number of rows that are having values in comparision to the threshold
    Symbol has the following:
    eq - equals
    ne - not equals'''
    if symbol == 'eq':
        return len(dataset[dataset[col]==threshold])
    elif symbol == 'ne':
        return len(dataset[dataset[col] != threshold])

def get_high_low(dataset,col):
    '''This gets the details of the highest and the lowest data of the specific column and dataset passed as argument'''
    #getting the value of highest data in col
    index_with_highest_value = dataset[col].idxmax()
    data_from_high_index = pd.DataFrame(movies_cleaned.loc[index_with_highest_value])
    
    #getting the value of lowest data in col
    index_with_lowest_value = dataset[col].idxmin()
    data_from_low_index = pd.DataFrame(movies_cleaned.loc[index_with_lowest_value])
    
    output = pd.concat([data_from_low_index,data_from_high_index], axis = 'columns')
    
    return output

def get_count_in_ascend(dataset,col,sepr = '|'):
    '''This function will count the number of times each values in the field occur and 
    returns a Series in ascending oder per the count'''
    data = dataset[col].str.cat(sep = sepr)
    data = pd.Series(data.split(sepr))
    return data.value_counts(ascending = False)

#To get the months with more profits, we need to group all of the months of years and then calculate the profits of those
#months giving a new dataframe which gives 'release-date' as index
profit_release = profit_movie.set_index('release_date')

#group all the data by month, since release date is in form of index, we extract month from it
groupby_index = profit_release.groupby([(profit_release.index.month)])

#Calculating the total profit per month
profit_per_month = groupby_index['profit_in_USD'].sum()

#converting table to a dataframe
profit_per_month= pd.DataFrame(profit_per_month)

#creating a list of months
month_list = ['January', 'February', 'March', 'April', 'May', 'June', 'July', 'August', 'September', 'October', 'November', 'December']
